# Temperature Converter

## 📌 Objective
Build a temperature converter web application that converts temperatures between Celsius, Fahrenheit, and Kelvin units.

## 🛠️ Tools Used
HTML5, CSS3, JavaScript

## 📋 Steps Performed
1. Created input field for entering temperature value
2. Added radio buttons to select input unit (Celsius, Fahrenheit, or Kelvin)
3. Implemented conversion formulas for all three temperature scales
4. Added convert button to trigger the conversion
5. Display converted results showing both other units
6. Added error handling for invalid inputs (empty field or non-numbers)
7. Made the design responsive for mobile devices
8. Added Enter key support for conversion

## 📊 Outcome
A fully functional temperature converter that accurately converts between Celsius, Fahrenheit, and Kelvin. The app shows error messages for invalid inputs and works smoothly on all devices.
